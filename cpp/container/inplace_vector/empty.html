std::inplace_vector<T,N>::empty

[edit template]
constexpr bool empty() const noexcept;
 (since C++26)

Checks if the container has no elements, i.e. whether begin() == end().

Contents

  • 1 Parameters
  • 2 Return value
  • 3 Complexity

[edit] Parameters

(none)

[edit] Return value

true if the container is empty, false otherwise.

[edit] Complexity

Constant.

[edit] Example

[edit]
Run this code
#include <cassert>
#include <inplace_vector>
 
int main()
{
    std::inplace_vector<char, 8> v;
    assert(v.empty());
 
    v.push_back('_');
    assert(not v.empty());
}
